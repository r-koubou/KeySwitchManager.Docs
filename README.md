# KeySwitchManager.Docs

[KeySwitchManager's](https://github.com/r-koubou/KeySwitchManager) document codes which powered by Sphinx.

## Requirements

- python 3.x
- pipenv

## Rebuild the environment

```bash
python -m pip install pipenv
pipenv install
```

## Build

```bash
cd docs/
pipenv run make html
```
