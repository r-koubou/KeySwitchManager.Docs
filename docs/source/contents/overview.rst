概要
========================================

キースイッチ情報をデータベースで管理し、対応したDAWのキースイッチ定義ファイルに相互変換可能なツール。

動作環境
--------------------------------------------

GUI
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

:OS: Windows 10
:ソフトウェア: `.net5 ランタイム <https://dotnet.microsoft.com/download>`_

コマンドライン
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

:OS: Windows, macOS, Linux
:ソフトウェア: `.net5 ランタイム <https://dotnet.microsoft.com/download>`_



対応DAW
----------------------------------------

- Cubase (Pro), Nuendo
- Studio One 5.2 以降
- Cakewalk
