Overview
========================================

キースイッチ情報をデータベースで管理し、対応したDAWのキースイッチ定義ファイルに相互変換可能なツール。

対応DAW
----------------------------------------

- Cubase (Pro), Nuendo
- Studio One 5.2 以降
- Cakewalk
